# Oricon

## Capabilities

Collect Japanese public Oricon ranking evidence and any values explicitly shown
on a cited public ranking page.

## Use when

Support a Japan-market ranking claim for a named Oricon chart and period, or
verify whether a public page visibly supplies a corresponding value.

## Do not use for

Inferring sales from rank, estimating unavailable values, or assuming a public
Oricon API exists.

## Access

Use public HTML pages under the [Oricon ranking index](https://www.oricon.co.jp/rank/)
through normal browser access. There is no assumed public API; respect page
access controls, robots directives, site terms, and rate limits.

## Inputs and outputs

Start with artist, title, chart category, and daily, weekly, monthly, or yearly
period. Retain the chart name, Japan market, period, rank, only values visibly
shown, page URL, edition/title, and observation date.

## Evidence

Authority tier: operator-published public ranking evidence for the exact chart
and period cited. Cite the direct Oricon page and distinguish a rank from any
separately displayed value.

## Limits and terms

The ranking index separates formats and reporting windows. A rank is not a
sales measurement, and values that are absent, paywalled, or not explicitly
shown must remain unavailable rather than inferred.

## Fallbacks

Use an official label or distributor press source for a specific announced
achievement, then [Web search](web-search.md) to locate primary evidence. State
that the Oricon value is unavailable when no public page supports it.

## Verification

Confirm chart category, period, edition, and artist before citation. Preserve
the Japan market and chart window, and do not compare ranks with sales or with
another chart's metric as a common unit.
